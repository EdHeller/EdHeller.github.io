Radio (2011)

A statically compiled dll reference to windows media player used in conjunction with RMTP streams. Simplistic and most web filters will ignore the connection for background music while I work. In the future I might package a different dll option or compile my own.
I should probably alter the transparency key among a few other modern fixes but at the time it was nice to play music on old or school workstations.

Fallout Style Terminal (2012)

A skinned terminal interface. I enjoy Fallout's old school aesthetic that monochromatic screen is a bit historic. Some older systems are more easy to work with when building hardware level interfaces (When hardware security was a write protection toggle switch) in that honor I stylized this terminal interface.
It functions as a normal CMD prompt using a pipe to file trick for showing any output.
